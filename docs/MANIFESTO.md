# Puntapie: Asistente De Estudio Personalizado

`fiuba-local-assistant` busca evolucionar de un asistente que responde sobre archivos locales a un tutor de estudio personalizado.

El objetivo es que cada alumno pueda integrar sus propios materiales, fechas, dudas y avances, y recibir una asistencia ajustada a como aprende mejor: que estudiar, como practicar, donde se esta trabando y que tan preparado esta para una evaluacion.

El producto no deberia ser solo un chat con PDFs. Deberia acompanarlo durante la cursada: planificar sesiones, corregir razonamientos, detectar errores recurrentes, generar checkpoints previos a examenes y convertir el progreso real en proximas acciones concretas.

Principio central:

> No estudia la materia en abstracto. Estudia con el alumno.
